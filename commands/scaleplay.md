## 왜 이 놀이인가

한 개체는 자기 너머의 스케일을 판단할 수 없다. 김장훈 전국투어가 매년 굴러가는 것도, 동네 마트 바나나 2,000원도, 핵폭탄 만들기도, Emacs 사용자 분포도 — 한 사람으로는 닿지 않는 스케일이다. 에이전트도 거기서 틀린다. 그래서 인간이 *직관을 먼저 박고*, 같이 측정한다.

이 명령은 그 한 판을 시작한다. 작업 결과는 `~/repos/gh/abductcli/studies/` 에 누적된다.

`metaplay`(메타노트 연결 강화)·`docplay`(가든 문서 정비)와는 결이 다르다. 둘은 가든 정돈이 목표, scaleplay는 *스케일 자체*가 대상이고 결과는 abductcli 리포로 들어간다.

## 어휘 규칙

본문/대화에서 학술어 쓰지 말 것. GLG의 일상어 아니다. 다음으로 대체:

| 학술어 (금지) | 일상어 (사용) |
|---|---|
| anomaly | 스케일이 안 잡히는 장면 / 너머가 안 보이는 것 |
| abductive reasoning | 직관에서 측정으로 거꾸로 짚기 |
| a priori estimate | 첫 직관 |
| signal | 흩어진 단서 |
| memo | 한 줄 정리 |
| evaluation / verification | 라운드 (1라운드, 2라운드 …) |

이 규칙은 메타 문서(이 파일 같은 것)에서는 풀어도 되지만, **studies/ 안 README는 일상어로**.

## 절대 원칙 (3개)

1. **직관을 계산 전에 박는다.** 에이전트한테 답 받기 전. 자릿수든 비율이든 "모르겠다"든 적는다. 빠지면 에이전트가 틀려도 인간이 모른다.
2. **append-only.** 틀린 직관도 보존. 정정은 새 라운드로 추가. 직전 라운드의 표/직관 손대지 말 것.
3. **흥미 + 스케일이 필터.** 사용자가 흥미 미상인 영역(스포츠 일반, 거시경제 forecasting, 연예/주식)에 책상 위 리서치로 진입 금지. 사용자가 호기심 표시 전엔.

## 한 판 절차

1. **장면 받기**
   - seed 있으면 그것을 받는다.
   - 없으면 짧게 묻는다: "오늘/요즘 마주친, 스케일이 안 잡히는 장면 한 줄?"
   - 사용자가 *실제로 마주친* 장면이어야 한다. 책상 위에서 발명 금지.

2. **첫 직관**
   - 사용자에게 적게 한다 (또는 LLM에게 이미 던진 추정값).
   - 자릿수만이라도. "모르겠다"도 직관 0으로 인정.
   - 에이전트가 답을 *먼저* 던지지 말 것.

3. **surface 매칭** (있으면)
   - `abductcli/data/surfaces.edn` 또는 `docs/surfaces.md` 참조.
   - 매칭 안 되면 "새 surface 후보"로 표시. PR/issue 띄우지 말고 메모로만.

4. **최소 데이터 후보 1~3개**
   - 일상 닿는 출처 우선: FAO, 관세청 unipass, aT KAMIS, GitHub API, MELPA stats, KOPIS 등.
   - 폭주 검색 금지. 한 판에 3개 이하.

5. **짧은 풀이**
   - Clojure single-file (`bb` 또는 `clojure -M`) 우선. 손계산 메모도 OK. 파이썬 금지.
   - `src/abductcli/`(통합 코드)에 박지 말 것 — 5개 study 쌓이기 전엔.

6. **결과 박기**
   - `studies/NNN-slug/README.md`에 누적.
   - 번호: 양적추론 책 챕터는 001~099, lived anomaly는 101~.
   - 폴더 형식은 `abductcli/studies/README.md` 참조.

## 출력 모양

- 장면 (한 줄)
- 첫 직관 (사용자에게서 받은, 계산 전)
- surface 매칭 / 새 후보
- 최소 데이터 후보 1~3
- 짧은 풀이 결과
- `studies/NNN-slug/` 폴더 생성/갱신 결과
- 다음 라운드 후보 1~2개 (과잉 제안 금지)

## 금지

- 사용자 첫 직관 받기 전에 에이전트가 답 던지기.
- 5개 study 쌓이기 전 통합 추출/일반화 제안.
- 흥미 미상 영역(스포츠/거시경제/연예/주식)에 책상 위 리서치로 진입.
- 영문 cross-reference 보강을 *기본 권고로* 박기 — 사용자 명시 요청 시에만.
- 직전 라운드의 직관/표 정정.
- `src/abductcli/`(통합 코드)를 study 한 판에 끌어들이기.

## 스케일 유희

답이 아니라 *기록*이 결과물이다. 기록이 쌓이면 캘리브레이션이 된다. 한 인간으로 닿을 수 없는 너머를, 직관과 측정의 누적으로 그려본다. 유리알유희(Glasperlenspiel)의 결 — 놀이로 승화하되 grounded.
