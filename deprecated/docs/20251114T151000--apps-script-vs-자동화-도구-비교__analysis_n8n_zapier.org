** 🎨 각 도구의 강점

*** Apps Script: Workspace Native

*최고의 선택 조건*:
- ✅ Google Workspace 중심 (90%+)
- ✅ 코드 제어 선호
- ✅ 무료 원함
- ✅ Custom Functions 필요
- ✅ UI 확장 필요 (Sidebar, Menu)

** 💡 Apps Script의 독보적 기능

*** 1. Custom Functions (Zapier/n8n 불가)

#+begin_src javascript
function PROJECT_HOURS(name, days) {
  var cal = CalendarApp.getDefaultCalendar();
  var events = cal.getEvents(
    new Date(Date.now() - days * 24 * 60 * 60 * 1000),
    new Date()
  );

  return events
    .filter(e => e.getTitle().includes(name))
    .reduce((sum, e) =>
      sum + (e.getEndTime() - e.getStartTime()) / 3600000, 0
    );
}
#+end_src

*사용*:
#+begin_src text

|----------|---------------|

| PM Agent | =PROJECT_HOURS(A3, 7) |
#+end_src
