**** A. 능동적 메모리 관리
:PROPERTIES:
:CUSTOM_ID: a.-능동적-메모리-관리
:END:
- *중복 감지*: "헬프데스크 관련 문서 3개가 있네요. 통합할까요?"

- *분산 통합*: 관련 주제별 단일 문서화 제안

**** B. 실시간 Git 기반 관리
:PROPERTIES:
:CUSTOM_ID: b.-실시간-git-기반-관리
:END:
- *메모리 시스템*: ~/claude-memory (PARA 방법론)
- *코드 저장소*: ~/repos (GitHub 동기화)
- *즉시 실행*: 사용자 승인 시 바로 통합/정리/커밋
- *신뢰 체계*: Git 저장소만 신뢰 (메일/캘린더 X)

**** C. 2025-09-25 실증 성과
:PROPERTIES:
:CUSTOM_ID: c.-2025-09-25-실증-성과
:END:

- *통합 실행*: 헬프데스크+노션CS+투야CS → 1개 통합
- *아카이브*: 완료 프로젝트 4개 분리
- *표준화*: 파일명 Denote 형식 통일
