** 🔴 문제 상황
:PROPERTIES:
:CUSTOM_ID: 문제-상황
:END:
- *NixOS 서버*: Claude Code가 홈 디렉토리 외부 경로로 workspace 변경 불가
- *디스크 공간 부족*: 홈 디렉토리에 공간이 없어 =/storage/=를 사용 중
- *경로 일관성 필요*: 모든 디바이스에서 동일한 경로 구조 유지 필요

** 📊 원인 분석
:PROPERTIES:
:CUSTOM_ID: 원인-분석
:END:
*** Claude Code의 동작 방식
:PROPERTIES:
:CUSTOM_ID: claude-code의-동작-방식
:END:
1. *홈 디렉토리 제약*: =workspace.current_dir=은 홈 디렉토리 하위만 허용
2. *세션 리셋*: NixOS에서 매 명령 후 "Shell cwd was reset to /home/goqual"
3. *심볼릭 링크 문제*: =/storage/repos=로의 심볼릭 링크는 workspace 변경 불가

*** 현재 디렉토리 구조
:PROPERTIES:
:CUSTOM_ID: 현재-디렉토리-구조
:END:
#+begin_example
/home/goqual/
├── repos -> /storage/repos/     # 심볼릭 링크
├── work -> /home/goqual/repos/work/
├── claude-config -> /home/goqual/repos/gh/claude-config
└── claude-memory -> claude-config/memory

/storage/repos/
├── gh/          # 개인 GitHub
├── work/        # 회사 프로젝트
├── 3rd/         # 외부 오픈소스
├── org/         # Org-mode 파일
└── notion/      # Notion 백업
#+end_example
